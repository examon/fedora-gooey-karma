## About

fedora-gooey-karma provides similar functionality to the fedora-easy-karma via GUI. It also currently provides some extra info like: yum info, bodhi info, test cases, bugs.

## Dependences

    python
    python-pyside
    yum
    yum-utils
    bodhi-client
    python-fedora
    fedora-cert
